# Tema Asistente IBUX

Tema moderno y limpio para WordPress diseñado específicamente para mostrar el Asistente Virtual IBUX.

## Instalación

1. El tema ya está instalado en `wp-content/themes/asistente-ibux/`
2. Ve a WordPress Admin → Apariencia → Temas
3. Activa el tema "Asistente IBUX"

## Configuración Inicial

Después de activar el tema, ejecuta el script de configuración para crear contenido de ejemplo:

1. Ve a WordPress Admin → Plugins → Editor de Plugins
2. O mejor, ejecuta este código desde el admin de WordPress (Tools → Run PHP Code, si tienes un plugin para eso)

O simplemente:
- Ve a Páginas → Añadir nueva y crea las páginas manualmente
- Ve a Apariencia → Menús y crea un menú
- Ve a Ajustes → Lectura y configura la página de inicio

## Características

- Diseño moderno y responsive
- Colores profesionales (gradientes púrpura/azul)
- Optimizado para mostrar el chat del asistente virtual
- Menú de navegación sticky
- Cards y secciones visualmente atractivas

## Personalización

Edita `style.css` para cambiar colores, fuentes y estilos.
